📚 Desenvolvimento Web 
📌 Introdução

A disciplina de Desenvolvimento Web tem como objetivo apresentar os conceitos fundamentais relacionados à criação de aplicações e interfaces para a web. O curso combina princípios de design visual, programação e arquitetura de sistemas, permitindo que os alunos desenvolvam soluções digitais funcionais e atrativas. 

Aula 01 - Apresentação do Plano…

Durante o semestre, serão explorados conceitos técnicos, ferramentas modernas e metodologias de desenvolvimento utilizadas no mercado de tecnologia.

🎯 Objetivos da Disciplina

A disciplina busca desenvolver nos alunos competências relacionadas ao desenvolvimento de aplicações web, incluindo:

Compreensão da arquitetura de aplicações web

Desenvolvimento de interfaces e layouts

Uso de ferramentas modernas de desenvolvimento

Aplicação de boas práticas de programação

Criação de projetos práticos e portfólio profissional

Além disso, o curso incentiva o trabalho em equipe e a construção de projetos que simulem desafios reais do mercado.

🧠 Conteúdos Abordados

Durante a disciplina serão estudados diversos conceitos importantes do desenvolvimento web.

🌐 HTML e CSS

Os alunos aprenderão os fundamentos de:

Estruturação de páginas web utilizando HTML

Estilização e design com CSS

Criação de layouts e organização de elementos na interface

Uso de seletores e propriedades de estilo

Essas tecnologias formam a base do desenvolvimento front-end.

💻 JavaScript

O JavaScript é uma linguagem de programação interpretada essencial para o desenvolvimento web. Ele permite:

Manipular elementos da página

Criar interatividade nas interfaces

Realizar comunicação com servidores

Implementar lógica de funcionamento da aplicação

O curso inicia com Vanilla JavaScript, ou seja, JavaScript puro, sem bibliotecas externas.

🧩 Frameworks Front-End

Frameworks facilitam o desenvolvimento de aplicações modernas. Serão apresentados conceitos relacionados a:

Componentização

Organização do código

Gerenciamento de estado

Integração com APIs

Alguns exemplos de frameworks estudados:

React

Vue

Angular

⚙️ Frameworks Back-End

No lado do servidor, frameworks ajudam na construção de aplicações web mais estruturadas.

Serão abordados conceitos como:

Arquitetura MVC

Desenvolvimento de APIs REST

Integração com bancos de dados

Autenticação e segurança de usuários

🗄️ Banco de Dados NoSQL

Bancos de dados NoSQL são sistemas não relacionais que permitem maior flexibilidade na estrutura dos dados.

Principais características:

Estrutura de dados dinâmica

Alta escalabilidade

Distribuição de dados em múltiplos servidores

Eles são amplamente utilizados em aplicações modernas e sistemas de grande escala.

🔄 Controle de Versão e Deploy

Outro conceito importante apresentado na disciplina é o controle de versão, que permite registrar e acompanhar alterações no código ao longo do tempo.

Entre os principais benefícios:

Recuperação de versões anteriores

Trabalho colaborativo em equipe

Desenvolvimento paralelo com branches

Ferramentas utilizadas:

Git

GitHub

GitLab

Bitbucket

Também serão abordados conceitos de deploy, que consiste em publicar uma aplicação em um servidor para que ela fique disponível online.

🏗️ Arquitetura de Projetos Web

Antes de desenvolver um sistema, é importante definir:

Escopo do projeto

Estrutura da aplicação

Prototipação da interface

Experiência do usuário (UX/UI)

Esse processo ajuda a validar ideias, melhorar a usabilidade e garantir que o projeto atenda às necessidades dos usuários.

🧰 Website Builders e CMS

Ferramentas conhecidas como Website Builders permitem criar sites sem a necessidade de programação avançada.

Vantagens

Facilidade de uso

Desenvolvimento rápido

Templates prontos

Baixo custo inicial

Limitações

Flexibilidade reduzida

Possível perda de performance

Dificuldade de migração para outras plataformas

Além disso, sistemas CMS (Content Management System) permitem gerenciar conteúdo de forma mais simples.

📊 Critérios de Avaliação

A avaliação da disciplina é composta por diferentes etapas:

Avaliação	Peso
Avaliação Continuada 1 (AC1)	15%
Avaliação Continuada 2 (AC2)	30%
Avaliação Geral	10%
Avaliação Final	45%

AC1: prova teórica de múltipla escolha

AC2: apresentação de projeto desenvolvido em grupo

Avaliação Final: documentação completa do projeto, semelhante a um mini TCC.

👥 Projeto em Grupo

Os alunos deverão formar grupos de 3 a 5 integrantes para desenvolver um projeto ao longo do semestre.

O grupo deverá:

Criar um repositório no GitHub

Registrar as atividades da disciplina

Desenvolver um projeto web

Documentar o desenvolvimento do sistema

📂 Atividade da Aula 01

Como primeira atividade, o grupo deve:

Criar um repositório no GitHub

Criar um arquivo Markdown (.md)

Escrever um resumo da Aula 01

Analisar 3 sites da internet

Identificar as tecnologias utilizadas nesses sites

Criar uma representação visual utilizando Miro

Salvar a imagem no repositório

💼 Importância do Portfólio

Manter um portfólio de projetos é fundamental para estudantes de tecnologia, pois:

Demonstra habilidades práticas

Mostra domínio de ferramentas de desenvolvimento

Facilita a avaliação por professores e recrutadores

Aumenta oportunidades de estágio e emprego

Além disso, ajuda no desenvolvimento de soluções reais e na preparação para desafios do mercado profissional.
